# Unit Economics — Примеры расчётов

## Формулы

- **LTV** (Lifetime Value) = Средний чек × Кол-во покупок за жизнь клиента
- **CAC** (Customer Acquisition Cost) = Затраты на маркетинг / Кол-во новых клиентов
- **LTV/CAC** > 3 = хороший бизнес
- **Payback period** = CAC / (Месячный доход с клиента)
- **Маржа** = (Цена - Себестоимость) / Цена × 100%

## Модели монетизации для контент-криейторов

### 1. Цифровой продукт (курс/гайд)
```
Цена: $49
Себестоимость: ~$0 (уже записано)
Маржа: ~95%
CAC (органика YouTube): $0-5
LTV/CAC: 10-50x
Для $10K/мес: 204 продажи
При конверсии 2% нужно: 10,200 визитов на лендинг
```

### 2. Подписка (Telegram Premium/Community)
```
Цена: $9/мес
Churn: 10%/мес
LTV: $9 × 10 мес = $90
CAC (органика): $0-3
LTV/CAC: 30x+
Для $10K/мес: 1,111 активных подписчиков
```

### 3. Шаблоны/Инструменты (одноразовая покупка)
```
Цена: $29 (базовый) / $99 (пакет)
Себестоимость: ~$0
Средний чек: $49
CAC: $0-5 (YouTube/Telegram)
Для $10K/мес: 204 продажи
При конверсии 3%: 6,800 визитов
```

### 4. SaaS (AI-инструмент по подписке)
```
Цена: $19/мес
Себестоимость API: $2-5/мес на клиента
Маржа: 74-89%
Churn: 8%/мес
LTV: $19 × 12.5 = $237
CAC: $10-20
LTV/CAC: 12-24x
Для $10K/мес: 526 клиентов
```

## Красные флаги

- Маржа < 50% → скорее всего не окупится
- CAC > $50 при цене < $100 → нужен другой канал
- LTV/CAC < 3 → убыточный бизнес
- Для выхода на $10K нужно > 1000 продаж/мес при нулевой базе → нереально быстро
- Churn > 15%/мес → продукт не решает боль
